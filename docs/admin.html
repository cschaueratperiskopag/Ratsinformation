Ersteinrichtung (einmalig)

Damit die Einstellungen gespeichert werden koennen, braucht die Seite Zugriff auf euer GitHub-Repository.

Erstellen unter: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token.
Berechtigung: nur "repo" ankreuzen. Token wird nur lokal in deinem Browser gespeichert.

Schlagwoerter

Nach diesen Begriffen werden die Ratsinformationssysteme durchsucht. Klicke das X zum Entfernen, oder fuege unten neue hinzu.

Kommunen

Diese Kommunen werden gescannt. Bestehende entfernen oder neue hinzufuegen.

+ Neue Kommune hinzufuegen

URL finden: Googeln "[Stadtname] Ratsinformationssystem" oder "[Stadtname] Buergerinfo".
System: Im Zweifel "ALLRIS" waehlen – das nutzen die meisten Kommunen.

Aenderungen werden auf GitHub gespeichert und beim naechsten Scan verwendet.
